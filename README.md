## Youtube Downloader Script Instructions

1. Clone the repository
2. Get Youtube `cookies.txt` using browser extensions
3. `cd` into the directory path and execute `yt_download_manager.bat`
4. Provide the Youtube video or playlist URL
5. URL options:
   - Video: Download in SCRIPT_DIR
   - Playlist: Download in ../PLAYLIST_NAME_DIR
  
**FYI: The script only supports Youtube video downloads at the moment. New music providers will be added soon.**
